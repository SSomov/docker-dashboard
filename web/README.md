# Docker Dashboard Web

Svelte приложение для отображения информации о Docker контейнерах.

## Установка зависимостей

```bash
bun install
```

## Разработка

```bash
bun run dev
```

## Сборка для продакшена

```bash
bun run build
```

Собранные файлы будут находиться в директории `public/`, которая используется Go сервером для раздачи статических файлов.
